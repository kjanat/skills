# Positive Lightning, Upward Lightning & Triggered Lightning

## Positive CG Lightning

### Fraction of CG Flashes

- Global average: <10%
- Summer US: ~3-5%
- Winter US (Jan-Feb): up to 25-80%
- Winter Japan (sea coast): up to 60%

### Conditions Conducive to Positive CG

| Condition                       | Mechanism                             |
| ------------------------------- | ------------------------------------- |
| Dissipating stage               | Main neg removed; pos exposed         |
| Winter storms                   | Positive charge displaced by shear    |
| Shallow clouds / MCS stratiform | Neg flashes decrease with cloud depth |
| Severe storms                   | Pos can dominate >30 min              |
| Forest fires / smoke            | 25-67% positive in affected regions   |

### Current Parameters (Berger et al. 1975, n=26)

| Parameter              | 95%      | 50%          | 5%           |
| ---------------------- | -------- | ------------ | ------------ |
| Peak current (kA)      | 4.6      | **35**       | **250**      |
| Charge (C)             | 20       | **80**       | 350          |
| Max dI/dt (kA/us)      | 0.20     | **2.4**      | 32           |
| Action integral (A^2s) | 2.5x10^4 | **6.5x10^5** | **1.5x10^7** |
| Flash duration (ms)    | 14       | **85**       | 500          |

5% peak current 250 kA vs 80 kA for negative → factor >3.
Highest directly measured: ~300 kA (Japan: 340, 320, 280 kA).

### Five Properties of Positive Lightning

1. **Single-stroke**: usually 1 stroke/flash (vs 3-5 for negative)
2. **Continuing current**: always present; can reach tens of kA
3. **Preceded by IC activity**: >100-200 ms before return stroke
4. **Long horizontal channels**: up to tens of km
5. **Leader mode**: can propagate continuously or stepped (unlike neg)

### Return-Stroke Speed

Positive: ~0.8x10^8 m/s. Negative: ~1.7x10^8 m/s (short segments).

## Upward Lightning

### Height Thresholds

| Object Height | Lightning Type                              |
| ------------- | ------------------------------------------- |
| <100 m        | Only downward                               |
| 100-500 m     | Both; upward fraction increases with height |
| >500 m        | Essentially only upward                     |

### Upward Leader Properties

| Type            | Speed (m/s)     | Step Length (m) | Interval (us) |
| --------------- | --------------- | --------------- | ------------- |
| Upward positive | 4x10^4-10^6     | 4-40            | 20-120        |
| Upward negative | 8x10^4-4.5x10^5 | 3-20            | 30-50         |

### Flash Parameters (Berger 1978, 50% values)

| Parameter             | Upward Neg (no RS) | Upward Neg (with RS) | Upward Pos |
| --------------------- | ------------------ | -------------------- | ---------- |
| Max IS current (A/kA) | 203 A              | 248 A                | 1.5 kA     |
| Charge (C)            | 12                 | 23                   | 26         |
| Duration (ms)         | 163                | 338                  | —          |
| % with return strokes | 20-50%             | —                    | —          |

### Current Reflections in Tall Objects

Lightning channel impedance: 600 ohm to 2.5 kohm.
Tower impedance: ~300 ohm.
Current at tower top NOT significantly affected by tower height.
Current at bottom contaminated by reflections.

## Rocket-Triggered Lightning

### Classical Triggering

- Rocket at ~200 m/s trailing grounded wire (~0.2 mm diameter)
- At 200-300 m: enhanced field → upward positive leader
- Wire vaporizes → ICC (hundreds of ms) → dart-leader/return-stroke sequences
- Ground E-field threshold: **4-10 kV/m** (Florida)
- Triggering potential at rocket altitude: **-3.6 MV**

### Return-Stroke Current (Triggered)

| Parameter         | Florida KSC | France   |
| ----------------- | ----------- | -------- |
| Median peak (kA)  | **12.1**    | **9.8**  |
| Max dI/dt (kA/us) | **91.4**    | **36.8** |
| n                 | 305 / 134   | 54 / 47  |

Triggered RS ≈ **subsequent strokes** in natural lightning.
GM peaks from 6 experiments: narrow range 9.9-15 kA.

### Key Finding: Grounding Independence

GM peak currents: **10-14 kA** regardless of grounding resistance
(0.1 ohm to 64 kohm). Lightning lowers its own grounding impedance via
fulgurites, surface arcing (up to 20 m), underground plasma channels.

### Altitude Triggering

Ungrounded wire + insulating gap → bidirectional leader:

- Positive charge up, negative charge down
- Produces first-stroke-like events (vs classical → subsequent-stroke-like)

### Unsuccessful Techniques (as of 2003)

- Laser beams (plasma sigma ~10^-3 S/m vs copper 5.8x10^7)
- Microwave beams
- Water jets
- Transient flames
