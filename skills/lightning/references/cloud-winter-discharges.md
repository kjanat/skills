# Cloud Discharges & Winter Lightning

## Cloud Discharge Types

1. **Intracloud** — within a single cloud (majority of cloud discharges)
2. **Intercloud** — between clouds
3. **Cloud-to-air** — between cloud and clear air

~75% of all lightning does NOT contact ground.
10+ cloud flashes may occur before first CG flash.

## Two-Stage Structure

| Stage                 | Duration     | Process                                          | Speed     |
| --------------------- | ------------ | ------------------------------------------------ | --------- |
| Early (active)        | Tens-100s ms | Negative channel extending (stepped-leader-like) | ~10^5 m/s |
| Late (final) / J-type | Tens-100s ms | Charge transport, K-processes                    | ~10^4 m/s |

Bidirectional leader: positive section in negative charge region, negative
section extending into positive region.

| Parameter         | Range                             |
| ----------------- | --------------------------------- |
| Duration          | 245-660 ms                        |

Spider lightning: extensive horizontal channels near cloud base.

## Narrow Bipolar Pulses / Compact IC Discharges

| Parameter          | Value                                 |
| ------------------ | ------------------------------------- |
| Risetime (10-90%)  | 2.3 us                                |
| Half-peak width    | 4.7 us                                |
| Duration           | 25.8 us                               |
| Peak (norm 100 km) | 9.5 V/m                               |
| HF radiation       | 10x typical lightning                 |
| VHF source power   | >100 kW (30 dB above other lightning) |
| Heights            | 8-11 km                               |

Responsible for trans-ionospheric pulse pairs (TIPPs).

## IC vs CG Comparison

| Feature                | Cloud                            | Ground    |
| ---------------------- | -------------------------------- | --------- |
| Leader current         | tens-hundreds A                  | Similar   |
| IB pulse duration      | 50-80 us                         | 20-40 us  |
| IB interpulse interval | 600-800 us                       | 70-130 us |
| Leader speed           | 1-3x10^5 m/s                     | Similar   |
| Low-origin flashes     | Indistinguishable from CG at VHF | —         |

## Winter Lightning (Japan)

### Formation

| Parameter       | Summer | Winter    |
| --------------- | ------ | --------- |
| Tropopause      | ~16 km | ~10 km    |
| Cloud base      | Higher | 200-800 m |
| Updraft at -10C | Higher | Very low  |

Siberian polar air over warm Tsushima current → convection.
Lightning does not extend >20-30 km inland.

### -10C Level Threshold

| Altitude   | Lightning         |
| ---------- | ----------------- |
| >1.8 km    | Relatively strong |
| 1.4-1.8 km | Weak or none      |
| <1.4 km    | None              |

### Unique Characteristics

| Feature                        | Value                |
| ------------------------------ | -------------------- |
| Upward flash fraction          | **98%**              |
| Positive CG fraction           | ~33% (vs 10% summer) |
| Bipolar fraction               | 5-33%                |
| Concurrent multi-tower strikes | up to 20%            |
| Charge transfer                | >1000 C reported     |
| Slowly varying currents        | Up to tens of kA     |
| Single-flash storms            | Common               |

### Triggered Winter Lightning

| Parameter            | Value                      |
| -------------------- | -------------------------- |
| Positive fraction    | 36-74%                     |
| Max negative peak    | -77 kA                     |
| Max positive peak    | +132 kA                    |
| Max negative charge  | -243 C                     |
| Max positive charge  | +640 C                     |
| Positive CC duration | >10 ms; avg current ~10 kA |

Mature stage lasts <10 min; graupel present for very short time.
Cloud evolution: developing (pos dipole) → mature (tripolar, <10 min) →
dissipating (pos monopolar, most of lifetime).
