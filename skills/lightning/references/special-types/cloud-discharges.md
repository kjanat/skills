# Cloud Discharges (Chapter 9)

## Overview

Cloud discharges (ICs) account for ~75% of all lightning. Include intracloud,
intercloud, and cloud-to-air. Much less studied than CG due to being hidden
within clouds.

## Overall Parameters (Table 9.1)

| Parameter                  | Value                                                  |
| -------------------------- | ------------------------------------------------------ |
| Total duration             | 200-500 ms                                             |
| Horizontal extent          | 5-20 km                                                |
| Vertical extent            | 3-6 km (connects negative and positive charge regions) |
| VHF source speed (initial) | 1-5x10^5 m/s                                           |

## Two Principal Types (VHF-UHF Imaging)

### Bilevel Flashes

- Connect main negative and upper positive charge regions
- Initial leader propagates upward from negative charge (at -10 to -25C)
- VHF sources move upward at 1-5x10^5 m/s
- Similar to CG first strokes but never reach ground

### Predominantly Horizontal Flashes

- Propagate primarily horizontally within single charge region
- More common in stratiform regions
- Can extend 50+ km horizontally

## Early (Active) Stage

- Duration: first ~100 ms
- VHF/UHF source rate: high (~10^3-10^4 sources/s)
- Main channel extension occurs during this stage
- Large bipolar E-field pulses occur (10-20 us width)

### Initial Breakdown Pulses

- Similar to those preceding first strokes in CG flashes
- Dominant polarity: opposite to CG (negative in physics convention)
- Largest pulses in a flash: 50-80 us total duration
- Interval: 600-800 us between pulses

## Late (Final) Stage

- Duration: remainder (~100-400 ms)
- K-type processes: transient rebrightening events
- Similar to J/K processes in CG flashes
- Intervals: ~5-20 ms between K-events
- VHF source rate decreases

## Narrow Bipolar Pulses (NBPs)

- Distinctive, isolated, very large-amplitude EM pulses
- Also called "compact intracloud discharges" (CIDs)
- Duration: 10-20 us
- Peak E-field: 5-10x stronger than normal IC pulses at same distance
- Intense VHF radiation (strongest of any lightning process)
- Occur at 8-15 km altitude (near cloud top)
- Both positive and negative polarities observed
- Mechanism: fast, short channel (~1 km) with high current (~30 kA)

## Comparison: Cloud vs Ground Discharges

| Feature                    | Cloud                      | Ground                      |
| -------------------------- | -------------------------- | --------------------------- |
| Fraction of all lightning  | ~75%                       | ~25%                        |
| Total duration             | 200-500 ms                 | 200-300 ms                  |
| Peak current (inferred)    | ~kA range                  | 30 kA (first stroke)        |
| Horizontal extent          | 5-50+ km                   | 1-10 km                     |
| EM radiation (HF-VHF)      | Stronger per flash         | Less HF/VHF                 |
| Associated thunder         | Less intense               | More intense (near channel) |
| Primary charge neutralized | Negative (between regions) | Negative (to ground)        |
